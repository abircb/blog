Dealing with dependancies in Python

DevTales
2019-08-13
Dependancy Management, Python, conda, pip

Application dependencies are libraries other than your project code that are required to create and run your application. When working on a collaborative project, managing dependancies becomes essential.

Here, I will show you how a couple of ways manage dependancies for a Python project.

Note: There is no right way to deal with dependancies for your project: it actually depends greatly on your project and how you decide to publish your software.
